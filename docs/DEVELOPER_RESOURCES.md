# Developer Resources: Campaign Chronicle Reader

This document serves as a living repository of documentation, reference material, and inspiration for developing the Obsidian Campaign Chronicle plugin.

## Official Obsidian API Documentation
*   **Main Developer Hub:** [Obsidian Developer Documentation](https://docs.obsidian.md/Home)
*   **Plugin Architecture:** [Anatomy of a Plugin](https://docs.obsidian.md/Plugins/Getting+started/Anatomy+of+a+plugin)
*   **User Interface (Views & Modals):** [User interface guide](https://docs.obsidian.md/Plugins/User+interface/Views)
*   **Working with the Workspace:** [Workspace API](https://docs.obsidian.md/Plugins/User+interface/Workspace)
*   **Reading/Writing Files:** [Vault API](https://docs.obsidian.md/Plugins/Vault)

## Community Resources & Best Practices
*   **Obsidian Sample Plugin:** [GitHub Repository](https://github.com/obsidianmd/obsidian-sample-plugin) (The boilerplate used for this repository).
*   **Obsidian API Type Definitions:** [obsidian.d.ts](https://github.com/obsidianmd/obsidian-api/blob/master/obsidian.d.ts) (Crucial for understanding available methods and properties).

## Similar / Reference Plugins
*As we research and develop, add links here to community plugins that implement similar features (e.g., custom views, continuous scrolling, or complex parsing).*

*   [Obsidian Book Navigator Plugin](https://github.com/jemberton/obsidian-booknav-plugin) - Adds bottom of page navigation links to previous and next pages.

*   [Obsidian Journaling Plugin](https://github.com/Ordeeper/obsidian-journaling-plugin) - Displays multiple pages of notes in a continuous view.

*   [Obsidian Yesterday Plugin](https://github.com/dominikmayer/obsidian-yesterday) - Displays notes in a continuous view.

*   [Obsidian Note ID Plugin](https://github.com/dominikmayer/obsidian-note-id) - Sets internal note id for keeping notes in a particular order.

*   [Obsidian Style Settings Plugin](https://github.com/obsidian-community/obsidian-style-settings) - The most extensive example of a plugin that uses the Style Settings API to allow users to customize the appearance of the plugin.

*   [Obsidian Dynamic Background Plugin](https://github.com/samuelsong70/obsidian-dynamic-background) - Example of setting a custom background image for the plugin.
